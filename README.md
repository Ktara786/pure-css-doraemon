# pure-css-doraemon

A fun CSS project that showcases my skills in creating graphics using only HTML and CSS. This project features a representation of Doraemon, the beloved robotic cat from the
cartoon.

## Technologies Used
- HTML
- CSS

## Features

- Pure CSS implementation
- Simple and clean code structure

## How to Download
- Click the green "Code" button on the repository page.
- Select "Download ZIP."
- Extract the ZIP file and open index.html in your web browser to View the Result. 

  Feel Free to Modify: You can edit the CSS styles in styles.css to customize the design to your liking!
